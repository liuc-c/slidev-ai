# Slidev 简介

专为开发者打造的演示文稿工具

# 什么是 Slidev？

- **基于 Markdown** - 使用你熟悉的 Markdown 语法编写幻灯片
- **开源且开发者友好** - 深度集成 Web 技术，支持 Vue 组件
- **专为程序员打造** - 代码高亮、实时预览、版本控制友好

---
layout: default
---

# 核心特性

- 📝 **Markdown 编写** - 像写文档一样写 PPT
- 🎨 **主题与样式系统** - 支持 UnoCSS / Tailwind CSS
- 📊 **图表与代码高亮** - 内置强大的代码演说能力
- 🎯 **导出为 PDF/PNG** - 甚至可以录制视频
- 🚀 **实时预览** - 即时反馈，高效创作

## 🙏 什么是弥勒佛？

弥勒佛是佛教中的**未来佛**，也是中国民间最为熟悉的佛教形象之一。

### 主要特征

- **慈眉善目** - 笑容可掬，给人温暖和亲切感
- **袒胸露腹** - 象征豁达、包容的境界
- **大肚能容** - "大肚能容，容天下难容之事"
- **笑口常开** - "笑口常开，笑天下可笑之人"

### 信仰意义

弥勒代表着**慈悲、喜乐和希望**，给人们带来正能量和美好的精神寄托。

# 北京历史文化景点

## 🏛️ 必游古迹

- **天安门广场** - 世界最大的城市广场，见证历史
- **故宫博物院** - 明清两代皇宫，世界文化遗产
- **颐和园** - 皇家园林，夏宫之美
- **天坛公园** - 祭天圣地，建筑杰作
- **长城** - 八达岭/慕田峪，世界奇迹
- **明十三陵** - 明代皇帝陵墓群
- **雍和宫** - 清代皇家寺院

> 💡 建议安排 2-3 天游览这些历史景点，感受古都魅力
